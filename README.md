webpack-react-demo
=====================

## Running Locally

1. You'll need `node` v8 and `yarn`.
2. Copy `config.template.json` to `config.json` and fill in all the variables.
3. Run this: 


```bash
npm install
npm run dev
```
or you can run this:

```bash
yarn
yarn dev
```

It should now be available at http://localhost:3000
